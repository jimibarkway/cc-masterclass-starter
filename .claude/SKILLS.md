# Skills used in this project

These are the external skills, gists and tools the Masterclass leans on. Most are installed ONCE, globally, during Modules 1 and 2, so they are already active in this project. You do not re-install them per repo.

The course Skills Vault is the single source of truth for full install steps and where each one is taught. This file is your quick reference.

## Installed globally in Modules 1 and 2 (already active here)
- **superpowers** (obra/superpowers): the brainstorm, plan, build, verify loop. A Claude Code plugin, follow its README. Your biggest safety net.
- **impeccable** (pbakaus/impeccable): the anti-AI-slop frontend skill (`/audit`, `/polish`). Follow its README.
- **last30days** (mvanhorn/last30days-skill): real-time market signal across Reddit, X, YouTube, HN and GitHub. Follow its README.
- **Direct-response copywriting skill:** saved to `~/.claude/skills/direct-response-copywriting.md`. `./setup` fetches it for you if it is missing.

## Installed per project when you reach the relevant lesson
- **supabase/agent-skills:** `npx skills add supabase/agent-skills` (Module 3, before you build auth).
- **Stripe MCP and Supabase MCP:** ask Claude Code to install them (Module 2).
- **Stripe CLI:** `brew install stripe/stripe-cli/stripe` on Mac (Module 3, for local webhook testing).

For full descriptions, Looms, and the rest of the toolkit, see the Skills Vault lesson in the classroom.
